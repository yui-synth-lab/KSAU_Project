## 2. File Structure & Descriptions

### Core Analysis (The Physics Engine)
*   **`ksau_config.py`**: The central configuration file defining physical constants ($\kappa = \pi/24$, $\alpha_{geom}$), mass formulas, and file paths.
*   **`topology_official_selector.py`**: The definitive source of truth for Quark/Lepton topology assignments. Implements the **Global Flavor Fit** (Mass + CKM optimization).
*   **`paper_I_validation.py`**: The primary validation script for Paper I, verifying the Holographic Duality (Bulk vs Boundary scaling).
*   **`ckm_final_audit.py`**: Detailed audit of the CKM matrix using the **Unified Lagrangian (Tunneling + Entropy)** model.


### Visualization (Figures)
*   **`plot_topological_landscape.py`**: Generates the "Holographic Landscape" plot (Figure 1), visualizing the split between Quark (Volume) and Lepton (Complexity) scaling.
*   **`ckm_discovery_plot.py`**: Visualizes the correlation between CKM mixing angles and topological volume difference ($\Delta V$).
*   **`cabibbo_anomaly_plot.py`**: Focuses specifically on the high-precision prediction of the Cabibbo angle ($V_{us}$).
*   **`g_minus_2_analysis.py`**: Analyzes the geometric origin of the anomalous magnetic moment.

### Predictions & Extensions
*   **`prediction_2026.py`**: Calculates specific falsifiable predictions for LHC Run 4 (Neutrino Sum, Dark Matter candidates).
*   **`robustness_check.py`**: Performs sensitivity analysis on the master constant $\kappa$ to prove the solution is a local optimum.
*   **`verify_neutrino_math.py`**: Solves the transcendental equations governing the neutrino mass hierarchy.

### Utilities & Audit
*   **`ckm_final_audit.py`**: Detailed error logging for the CKM matrix to identify the "Tunneling Anomaly."
*   **`verify_neutrino_math.py`**: Mathematical scratchpad for neutrino scaling.

## 3. Dependencies

*   **Python 3.8+**
*   **NumPy**: Numerical operations.
*   **Pandas**: Data handling for KnotInfo/LinkInfo databases.
*   **Matplotlib**: Figure generation.
*   **SciPy**: Optimization and root finding (for neutrino equations).

Install requirements:
```bash
pip install numpy pandas matplotlib scipy
```

## 4. Data Sources
The code relies on the following data files in `../data/`:
*   `physical_constants.json`: Experimental values (PDG 2024) and KSAU theoretical constants.
*   `topology_assignments.json`: Generated by `topology_official_selector.py`.
*   `knotinfo_data_complete.csv`: External database of knot invariants.
*   `linkinfo_data_complete.csv`: External database of link invariants.
