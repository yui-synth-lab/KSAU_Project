## 仮説 H46: 10D Compactification Gravity Precision

### 帰無仮説 (H0)
10Dコンパクト化パラメータ（オイラー標数等）に基づく補正モデルは、現在のKSAU重力定数予測誤差（0.0815%）を有意に改善しない。

### 対立仮説 (H1)
10次元バルクの幾何学的・位相的特徴（カラビ＝ヤウ多様体やD4格子から第一原理で導出される不変量）を組み込んだ非摂動モデルは、重力定数Gの実験値に対する相対誤差を恣意的なパラメータ追加なしに削減する。

### データ要件
SSoTの `gravity`（`G_newton_exp`等）、`dimensions` セクションのデータ群。実データのみ使用。

### 物理的制約（PHYSICAL CONSTRAINTS）
- **適用範囲:** GRAVITY_SECTOR（重力結合定数Gの計算）
- **最大自由パラメータ数:** 0
- **導出要件:** FIRST_PRINCIPLES（任意の多項式フィッティング禁止。補正項はコンパクト化空間の位相幾何学的性質から厳密に導出されること。）

### 統計的有意性基準
- Bonferroni補正後閾値: 0.016666
- FPR（モンテカルロ置換検定）: < 50%

### 撤退基準（削除不可）
- Bonferroni補正後 p > 0.016666 → 即座に REJECT
- FPR > 50% → 即座に REJECT
- 物理的制約を満たさないモデルを提出した場合 → 即座に MODIFY
- 5 イテレーション到達で進展なし → REJECT
- Reviewer の連続 STOP 判定 2回 → 強制終了

### テスト手法
導出されたG理論値とG_newton_expの相対誤差評価、およびモンテカルロ法による代替コンパクト化パラメータ群（無作為な次元・位相幾何学数）との比較検定。

### 最大イテレーション数
5

## 仮説 H47: Independent Regression Validation of κ via V_eff

### 帰無仮説 (H0)
全フェルミオン質量（9点）と有効体積 V_eff に対する単回帰分析（ln m_obs = κ_fit * V_eff + C）において得られる推定量 κ_fit の95%信頼区間は、理論値 π/24 を含まない。

### 対立仮説 (H1)
V_eff を独立変数とする回帰推定量 κ_fit の95%ブートストラップ信頼区間は理論値 π/24 (約 0.1309) を厳密に含み、第一原理からの導出を統計的に支持する。

### データ要件
SSoTの `particle_data`（フェルミオン9個の観測質量）、`effective_volume_model`から算出される V_eff。実データのみ使用。

### 物理的制約（PHYSICAL CONSTRAINTS）
- **適用範囲:** ALL_9_FERMIONS
- **最大自由パラメータ数:** 0
- **導出要件:** STATISTICAL_VALIDATION（この仮説は回帰係数の検定であり、新たな補正項の導入を目的としない。κの固定値入力は厳禁。NEG-20260225-05の反省から純粋体積Vではなく有効体積V_effを用いる。）

### 統計的有意性基準
- Bonferroni補正後閾値: 0.016666
- FPR（モンテカルロ置換検定）: < 50%

### 撤退基準（削除不可）
- Bonferroni補正後 p > 0.016666 → 即座に REJECT
- FPR > 50% → 即座に REJECT
- 物理的制約を満たさないモデルを提出した場合 → 即座に MODIFY
- 3 イテレーション到達で進展なし → REJECT
- Reviewer の連続 STOP 判定 2回 → 強制終了

### テスト手法
ブートストラップ法（N=10000）によるκ推定量（κ_fit）の95%信頼区間の構築および、κ=π/24との等価性検定（One-Sample T-test または同等手法）。

### 最大イテレーション数
3

## 仮説 H48: Non-linear Topological Mass Correction

### 帰無仮説 (H0)
非線形トポロジカル補正項（例：指数関数的トーション結合など）は、ベースライン（κ=π/24, V_eff）の予測精度を有意に（Bonferroni補正後閾値未満で）向上させない。

### 対立仮説 (H1)
第一原理（高次元不変量や非線形ST）から導出される非線形結合項は、フェルミオン質量の予測残差を有意に削減し、FPR < 50% かつ p < 0.016666 を満たす。

### データ要件
SSoTの全フェルミオン質量および、結び目不変量 (V, ST, Det, n など)。実データのみ使用。

### 物理的制約（PHYSICAL CONSTRAINTS）
- **適用範囲:** ALL_9_FERMIONS
- **最大自由パラメータ数:** 1
- **導出要件:** FIRST_PRINCIPLES（カーブフィッティングのための恣意的な非線形関数の導入禁止。幾何学的に意味のある非線形結合であること。NEG-20260226-04 で線形ST補正が過学習で棄却されたため、線形結合アプローチは禁止。）

### 統計的有意性基準
- Bonferroni補正後閾値: 0.016666
- FPR（モンテカルロ置換検定）: < 50%

### 撤退基準（削除不可）
- Bonferroni補正後 p > 0.016666 → 即座に REJECT
- FPR > 50% → 即座に REJECT
- 物理的制約を満たさないモデルを提出した場合 → 即座に MODIFY
- 5 イテレーション到達で進展なし → REJECT
- Reviewer の連続 STOP 判定 2回 → 強制終了

### テスト手法
非線形回帰による残差削減率の評価、モンテカルロ置換検定によるFPRの算出。LOO-CVによる過学習チェック（Training MAEとLOO-MAEの乖離監視）。

### 最大イテレーション数
5
