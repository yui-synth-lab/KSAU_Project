### 1.4 Phase 1 の結論

**v4.1 目標（MAE < 5%, Max < 10%）は、単純な補正項の追加では達成不可能。**

理由:
1. **BIC でベースラインが最良** — パラメータ追加の改善が統計的に正当化されない
2. **補正パラメータに理論定数（G, π, 7, 9, 10）での表現が見つからない** — 全モデルで "No close match"
3. Bottom (-17.3%) と Down (+14.0%) の大誤差は**リンク候補選択（行列式規則の制約）**に起因し、公式の修正では解消できない
4. 最良の 1-param モデル（Quadratic V）でも Top を改善する代わりに Bottom が悪化（-17.3% → -18.1%）

**Phase 1 の戦略転換:**
- 公式修正ではなく、**行列式規則を満たす代替リンク候補の再探索**が必要
- または Chern-Simons 不変量（複素体積の虚部）による補正を検討
- v4.0 のゼロパラメータ公式を**維持**し、誤差は「離散的リンク選択の限界」として許容する選択肢もある

### 2.3 最有力候補の詳細

#### 候補 A: Twist Knot 補正（Model i）— 最もシンプル

$$\ln(m_l) = \frac{2}{9}G \cdot N^2 - \frac{1}{6}\mathbb{1}_{twist} + C_l$$

- **MAE = 0.4%**, Max = 1.2%（タウのみ）
- v4.0 構造を完全に保存
- δ = -0.1642 ≈ **-1/6**（誤差 0.0025、1.5% 以内）
- 物理的意味: twist knot は torus knot より「ねじれ」が少なく、質量が $e^{1/6} ≈ 1.18$ 倍軽くなる
- **問題:** -1/6 が正確に成立する理由の理論的根拠が必要

#### 候補 B: N² + ln(Det) 補正（Model e）— 最も数論的に美しい

$$\ln(m_l) = \alpha \cdot N^2 - \frac{1}{3}\ln(\text{Det}) + C$$

- 完全フィット（MAE = 0%）
- β = -0.3272 ≈ **-1/3**（誤差 0.006、1.8% 以内）
- α = 0.2108（(2/9)G = 0.2035 から 3.5% ずれ）
- **-1/3 は down-type quark の電荷と一致** — Quark-Lepton 統一の手がかり？
- **問題:** 3 データ点 / 3 パラメータ（過決定ではない）

#### 候補 C: N² + Genus 補正（Model b）— 最も物理的に透明

$$\ln(m_l) = \alpha \cdot N^2 + \beta \cdot g + C$$

- 完全フィット（MAE = 0%）
- α = 0.179, β = 0.492 ≈ **1/2**
- genus = {1, 2, 3}（電子、ミューオン、タウ）— 自然な整数列
- 物理的意味: genus（穴の数）が大きいほど質量が $e^{1/2} ≈ 1.65$ 倍ずつ増加
- **問題:** α が (2/9)G からかなりずれる（-12%）; β = 1/2 の根拠不明

### 2.4 Phase 2 の結論

**Muon 異常 (+17.8%) は複数の方法で完全に解決可能。**

推奨順位:
1. **Model (i) Twist correction**: v4.0 互換、δ ≈ -1/6、MAE = 0.4%
2. **Model (e) N² + ln(Det)**: β ≈ -1/3 が電荷と共鳴、MAE = 0%
3. **Model (b) N² + Genus**: 物理的に最も透明、MAE = 0%
