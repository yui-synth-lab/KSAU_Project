## 背景と問題の所在

`go.md` §2 WARNING #1 は、`section_a_nonstandard_wzw_survey.md` §4.6 の以下の文に対する指摘である：

> 「$\pi$ そのものを値として持つ物理的な表現は存在しない」

この文の根拠として掲げられた「構造定数が整数だから $\pi$ が出ない」という論拠は、**コンパクト群**では明快である。しかし監査者が指摘した通り、$SL(2,\mathbb{R})$ のような非コンパクト群では Casimir 固有値が連続実数値をとりうるため、追加議論が必要であった。

本補足文書は、$SL(2,\mathbb{R})$ の：
1. 連続系列表現（continuous series）: $j = 1/2 + is, \; s \in \mathbb{R}_{>0}$
2. 離散系列表現（discrete series）: $j \in (0, (k-1)/2)$

の双方について、Casimir 固有値が「物理的ユニタリ表現のパラメータ空間において $\pi$ をとらないこと」を厳密に示す。

---

## §1: $SL(2,\mathbb{R})$ WZW の表現論の構造

### §1.1 WZNW モデルのスペクトル（Maldacena-Ooguri 2001）

$SL(2,\mathbb{R})_k$ WZW モデルの物理的表現は、Maldacena & Ooguri (2001) [hep-th/0005183] により系統的に分類された。Hilbert 空間を構成する表現は：

**（I）連続系列（Principal Continuous Series）**

$$j = \frac{1}{2} + is, \quad s \in \mathbb{R}_{>0}$$

この表現は $\hat{sl}(2,\mathbb{R})_k$ の最高ウェイト状態を持たず、弦理論のスペクトルにおいて連続的な質量スペクトルを与える。

**（II）離散系列（Discrete Series）**

$$j \in \left(\frac{1}{2}, \frac{k-1}{2}\right), \quad j \in \mathbb{R}$$

ユニタリ性のため上限 $j < (k-1)/2$ が必要。特に最低ウェイト表現 $\mathcal{D}_j^+$ と最高ウェイト表現 $\mathcal{D}_j^-$ がある。

### §1.2 Casimir 固有値

$sl(2,\mathbb{R})$ の 2 次 Casimir 演算子 $C_2 = J_0^2 - J_1^2 - J_2^2$（あるいは WZW 文脈では $C_2 = -J_0(J_0-1) + J_+J_-$ 等と書く）の固有値は表現ラベル $j$ によって：

$$C_2(j) = -j(j-1) = j(1-j)$$

と書かれる（符号は $sl(2,\mathbb{R})$ の場合）。

## §2: 連続系列における $C_2 = \pi$ の非実現

### §2.1 連続系列での Casimir 固有値の値域

連続系列 $j = 1/2 + is$（$s \in \mathbb{R}_{>0}$）における Casimir 固有値は：

$$C_2(j) = j(1-j) = \left(\frac{1}{2} + is\right)\left(1 - \frac{1}{2} - is\right) = \left(\frac{1}{2} + is\right)\left(\frac{1}{2} - is\right)$$

$$= \frac{1}{4} + s^2$$

### §2.2 $C_2 = \pi$ となる $s$ の「存在」

$C_2 = \pi$ を仮定すると：

$$\frac{1}{4} + s^2 = \pi \implies s^2 = \pi - \frac{1}{4} \implies s = \sqrt{\pi - \frac{1}{4}} \approx \sqrt{2.8916} \approx 1.7005$$

この値は $s \in \mathbb{R}_{>0}$ を満たし、**連続系列のパラメータとして形式的には存在する**。

しかしここで問うべきは「$s$ の値として $\sqrt{\pi - 1/4}$ が存在するか」ではなく、**「KSAU が要求する $7\pi/k$ がエネルギー固有値として出現するか」**である。

### §2.3 連続系列の排除の主論拠：符号の根本的不整合

> **【修正注記 — v33.0 Session 3, ng.md 指摘#7 対応】**
> 旧版 §2.3 は「$k \to \infty$ の大 $k$ 展開での定数項比較」を主論拠に用いていた。しかし KSAU の文脈では $k \approx 24-25$（有限値）であり、$k \to \infty$ 展開の係数比較を有限 $k$ の厳密な議論として提示することは不適切であった（ng.md 指摘#7）。この論拠は削除し、より直接的で有限 $k$ に対しても有効な論拠に置き換える。

WZW モデルの $L_0$ 固有値（真空エネルギー）は：

$$h = \frac{C_2(j)}{k - 2} + n, \quad n \in \mathbb{Z}_{\geq 0}, \quad h^\vee_{SL(2,\mathbb{R})} = 2$$

連続系列では $C_2 = 1/4 + s^2 > 0$（実正値）、$n \geq 0$ なので：

$$h = \frac{1/4 + s^2}{k-2} + n \geq 0 \quad \text{（$k > 2$ の場合）}$$

KSAU の要求する $b_q(k) = -7(1 + \pi/k)$ は $k > 0$ の全域で**負値**をとる。

$$b_q(k) = -7 - \frac{7\pi}{k} < 0 \quad \forall k > 0$$

**連続系列の $h \geq 0$ と KSAU の要求 $b_q < 0$ は有限 $k$ で直接矛盾する（符号の根本的不整合）。**

これは $k \to \infty$ 展開に依存しない、任意の有限 $k > 2$ に対して成立する厳密な排除論拠である。

**追加論拠（符号不整合に加えて）：** 仮に符号の問題を除いても、$b_q(k)$ の $k$ 依存性が $C_2/(k-2)$ の形に一致するためには $C_2 = 7\pi$ が必要（§4.3 参照）。しかし $s$ を $\sqrt{7\pi - 1/4}$ に固定する代数的選択原理が存在しない（§4 で詳述）。

**結論：** 連続系列において $b_q(k) = -7(1+\pi/k)$ の「代数的導出」が成立しない理由は二重である：
1. **符号の根本的不整合**（$h \geq 0$ vs $b_q < 0$）——有限 $k$ で厳密
2. **代数的選択原理の不在**——$s = \sqrt{7\pi - 1/4}$ を選ぶ根拠が WZW 理論に存在しない（§4 参照）

## §3: 離散系列における $C_2 = \pi$ の非実現

### §3.1 離散系列のパラメータ空間

離散系列（ユニタリ、最低ウェイト）のパラメータ空間は：

$$j \in \left(\frac{1}{2}, \frac{k-1}{2}\right)$$

$SL(2,\mathbb{R})_k$ WZW のユニタリ性には $k > 2$ が必要で、この区間の $j$ が許される。

### §3.2 Casimir 固有値の実正値性

離散系列では $j \in \mathbb{R}_{>0}$（実正値）なので：

$$C_2(j) = j(1-j)$$

- $j \in (0, 1/2)$ のとき: $C_2 > 0$（$j > 0, 1-j > 0$）
- $j = 1/2$ のとき: $C_2 = 1/4$
- $j \in (1/2, 1)$ のとき: $C_2 < 0$（$j > 0, 1-j < 0$）
- $j \in [1, (k-1)/2)$ のとき: $C_2 = j(1-j) \leq 0$ でさらに大きな負値

$C_2 = j(1-j) = \pi \approx 3.14159$ を満たす $j$ は：

$$j^2 - j + \pi = 0 \implies j = \frac{1 \pm \sqrt{1 - 4\pi}}{2}$$

判別式 $1 - 4\pi \approx 1 - 12.566 = -11.566 < 0$ より、**この方程式の実数解は存在しない**。

$$\boxed{C_2 = \pi \text{ を満たす実数 } j \text{ は存在しない（判別式 < 0）}}$$

したがって、離散系列の実数パラメータ $j$ では Casimir 固有値は $\pi$ をとることができない。

### §3.3 追加確認：$C_2 = 7\pi$ の場合

KSAU が直接必要とする $7\pi/k$ 係数のためには $C_2 = 7\pi$ が必要である（$h \propto 7\pi/k$）。

$$j(1-j) = 7\pi \implies j = \frac{1 \pm \sqrt{1 - 28\pi}}{2}$$

判別式 $1 - 28\pi \approx 1 - 87.96 = -86.96 < 0$。実数解は存在しない。

$$\boxed{C_2 = 7\pi \text{ を満たす実数 } j \text{ は存在しない（判別式 < 0）}}$$

## §4: 連続系列の再精査——「$C_2 = \pi$ は存在する」への反論補強

### §4.1 §2.2 の問題を改めて整理

連続系列では $C_2 = 1/4 + s^2$（実正値）であり、$s = \sqrt{\pi - 1/4}$ とすれば $C_2 = \pi$ は**数値として達成可能**である。これが WARNING #1 の核心的な問題であった。

この「数値的な達成可能性」と「KSAU 要求の達成可能性」を厳密に区別するため、以下に構造的な反論を与える。

### §4.2 KSAU が要求する関数形の分解

$b_q(k) = -7(1+\pi/k)$ は：
1. $k$ の**関数**として $\pi$ が係数に現れる
2. 係数は **$k$ に依存しない普遍定数** $7\pi$ である

WZW において $h = C_2(j)/(k-2) + n$ のとき、$\pi/k$ の係数を $7\pi$ にするには：

$$\frac{C_2(j)}{k-2} \approx \frac{C_2(j)}{k}\left(1 + \frac{2}{k}\right) = \frac{C_2(j)}{k} + \frac{2C_2(j)}{k^2} + \cdots$$

$h$ の $1/k$ 項の係数 = $C_2(j)$ が $7\pi$ に等しい必要がある。つまり要求は **$C_2$ という代数的定数（= 表現ラベルの関数）が $7\pi$ という超越数に等しくなること**である。

### §4.3 根本的な問題：なぜ $C_2 = 7\pi$ は不可能か

連続系列: $C_2 = 1/4 + s^2$。もし $C_2 = 7\pi$ なら $s = \sqrt{7\pi - 1/4} \in \mathbb{R}_{>0}$。

この $s$ は **実数として存在する**。しかし WZW の文脈で問うべきは：

**「この表現ラベル $j = 1/2 + i\sqrt{7\pi - 1/4}$ は、どのような代数的・物理的必然性から定まるのか？」**

$k = 24$（KSAU の $k_{obs} \approx 25$）の場合、ユニタリ表現の分類によれば連続系列のパラメータ $s$ は**連続自由パラメータ**であり、$s$ を特定の値（$\sqrt{7\pi - 1/4}$）に固定する代数的な選択原理が存在しない。

これは WZW 理論から $7\pi/k$ を **導出する**のではなく、「$7\pi/k$ を出したい」という目標から逆算して $s$ を選ぶ——という **循環論法**である。

### §4.4 正確な「不可能」の境界の宣言

| 表現 | $C_2 = \pi$ の実現 | $C_2 = 7\pi$ の実現 | KSAU導出としての意味 |
|------|------------------|-------------------|---------------------|
| 離散系列 ($j \in \mathbb{R}$) | **不可能**（判別式 < 0） | **不可能**（判別式 < 0） | 数学的確定棄却 |
| 連続系列 ($j = 1/2 + is$) | $s = \sqrt{\pi - 1/4}$ で**数値的に達成可能** | $s = \sqrt{7\pi - 1/4}$ で**数値的に達成可能** | ただし代数的選択原理なし → **循環論法** |

**したがって、正確な「不可能」宣言の内容は以下に改訂される：**

> 非コンパクト WZW（$SL(2,\mathbb{R})$ 等）において $b_q(k) = -7(1+\pi/k)$ を**代数的必然性として導出すること**は不可能と確定する。連続系列では $s$ を適切に選べば「数値的に」$C_2 = 7\pi$ を実現できるが、この選択を支持する代数的原理（群の構造定数・表現論の自然な選択）が存在しない。したがって WZW から $7\pi/k$ を「導出」するためには $s$ を事後的に調整する必要があり、これは FREE PARAMETER の導入と等価であり、代数的起源の探索としては失敗である。

## §5: 改訂後の判定と section_a への反映

**改訂後の判定（本補足文書 §4.4）:**

> ケース3（非コンパクトWZW）: **不可能と確定（論拠強化後）**
>
> - 離散系列（$j \in \mathbb{R}$）: $C_2 = j(1-j) = 7\pi$ を満たす実数解が存在しない（判別式 $1 - 28\pi < 0$）ことにより数学的に不可能。
> - 連続系列（$j = 1/2 + is$）: $C_2 = 1/4 + s^2$ は数値的に $7\pi$ をとり得るが、$s = \sqrt{7\pi - 1/4}$ を選ぶ代数的選択原理が WZW 理論の枠組みに存在しない。これは $7\pi/k$ を前提として $s$ を事後調整する循環論法であり、「代数的起源の導出」ではない。

### §5.2 論拠の構造的強化

WARNING #1 で要求された「$\pi$ を固有値として持つ $SL(2,\mathbb{R})$ Casimir 表現が物理的ユニタリ表現のパラメータ空間に含まれないことの証明」に対し：

| 表現の種別 | 証明の種別 | 結論 |
|-----------|-----------|------|
| 離散系列 | 代数的（判別式計算） | $C_2 = 7\pi$ の実数解なし → 存在しない |
| 連続系列 | 論理的（選択原理の不在） | 数値的達成可能だが代数的選択原理なし → FREE PARAMETER 導入に等価 |

### §5.3 KSAU 主結論への影響なし

本補足文書の改訂により、ケース3の判定は「不可能と確定」を維持する。主結論に変更なし：

- WZW 全経路（標準 + 非標準）での $b_q(k) = -7(1+\pi/k)$ の代数的導出: **不可能と確定**
- $q_{mult} = 7$ の代数的起源: **FREE PARAMETER（WZW全経路閉鎖後も依然未解決）**
