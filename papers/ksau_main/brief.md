## 3. 必ず言及すべき否定的結果

**論文の信頼性は、以下の否定的結果を正直に記述できるかにかかっている。**

| ID | 否定的結果 | 論文での扱い |
|----|-----------|-------------|
| H33 | κ は質量回帰から独立に回復できない (CI excludes π/24) | Discussion: κ は入力パラメータであり、データから導出される量ではない |
| H47 | V_eff 回帰が κ を 11 倍過大評価 | Discussion: V_eff は代理変数であり独立変数ではない |
| H58 | 3予測値の同時精度がランダムと区別不能 (p=0.067) | Discussion: 個別予測は有意だが joint test は未達 |
| H59 | トーション補正の LOO 不安定性 (LOO-R²=0.11) | Discussion: ST 補正は数学的に魅力的だが汎化しない |
| H60 | det ≡ 0 (mod 24) が安定性と負の相関 | Discussion: 24-cell 対称性の予測に反する結果 |
| H23 | 位相離散化 K=24 がトートロジカル (FPR=93.82%) | Methods: K=24 は導出結果であり検証対象ではない |

## 5. 論文構成の指針

### Abstract (150語以内)
- KSAU フレームワークの概要（1文）
- 主要結果3つ（質量公式 R², 割当一意性, G 導出精度）
- 検証規模（69仮説テスト、26サイクル）
- 否定的結果への一言言及（「複数の仮説が棄却された」）

### 1. Introduction
- 粒子物理と結び目理論の既存の接点（Witten, Atiyah）
- KSAU の位置づけ: 「予測フレームワーク」ではなく「データ駆動相関の体系的検証」
- 本論文の貢献: 69仮説の体系的テストによる肯定・否定結果の包括的記録

### 2. Theoretical Framework
- 3多様体と双曲体積の基礎
- κ = π/24 の導出（24-cell 共鳴条件）
- 質量公式 ln(m) = ηκV + corrections の構造
- トポロジー割当ルール（Brunnian/Borromean 条件）

### 3. Methods
- AIRDP プロトコルの説明（Researcher → Reviewer → Judge）
- SSoT による定数管理
- 統計的手法（Bonferroni, LOO-CV, Monte Carlo FPR）
- 使用データ: KnotInfo/LinkInfo（crossing 3-12, 6502 結び目）

### 4. Results
- 4.1 質量公式の検証
- 4.2 トポロジー割当の一意性
- 4.3 重力定数の導出
- 4.4 CKM 行列の再現
- 4.5 量子数の幾何学的起源
- 4.6 否定的結果の要約表

### 5. Discussion
- 理論の限界と未解決問題
- κ の循環性問題（H33, H47）
- joint prediction の失敗（H58）
- トーション補正の過学習（H59）
- 検証可能な予測の不在
- 他の TQFT アプローチとの比較

### 6. Conclusion
- 慎重な要約（「証明した」ではなく「体系的な相関を確認した」）
- 今後の課題

### Appendix
- A: 全69仮説の一覧表（ID, 名称, 判定, 主要指標）
- B: トポロジー割当表（12粒子 × 結び目不変量）
- C: SSoT 定数の完全リスト
