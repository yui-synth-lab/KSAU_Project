## 概要 (Abstract)
我々は、標準模型におけるフェルミオンの質量が、3つの独立した選択則を通じて双曲結び目および絡み目（リンク）の不変量によって決定されていることを示す統計的かつ幾何学的な証拠を提示する。クォークとレプトンに関連付けられた絡み目補空間（Link Complement）の位相的性質を解析することで、以下の3つの原理を特定した：(1) 粒子の閉じ込めと絡み目の成分数（Component Number）を関連付ける**「閉じ込め・成分数対応原理 (Confinement-Component Correspondence)」**、(2) 電荷のパリティと結び目行列式（Knot Determinant）を結びつける**「電荷・行列式法則 (Charge-Determinant Law)」**、および (3) カタラン定数（$G$）と有効高次元構造（$D \sim 10$）によって支配される統一的な**「幾何学的質量スケーリング (Geometric Mass Scaling)」**である。提案されたモデルは、6種類のクォークと3種類の荷電レプトンすべての質量を、**7.9%**の全体平均絶対誤差（MAE）（クォーク: 8.7%、レプトン: 6.3%）で適合させる。クォークチャネルでは**連続的な調整パラメータを一切使用せず**（ゼロ・パラメータ）、レプトンチャネルでは電子の質量によって固定される1つの規格化定数（$C_l$）のみを使用する。この枠組みは、$p < 10^{-5}$ という統計的有意性を達成している。

## 1. 序論 (Introduction)

### 1.1 階層性問題
素粒子の標準模型（SM）は、基本的な相互作用の記述において目覚ましい成功を収めてきたが、フェルミオン質量の起源については未解決のままである。観測される質量は、電子（0.511 MeV）からトップクォーク（173 GeV）まで5桁以上にわたり、SMの枠組みの中では恣意的に見える階層構造を示している [1]。これらの質量を決定する湯川結合定数は、ラグランジアンにおける自由パラメータであり、「なぜ」これらの特定の値が存在するのかについての洞察を与えない。

### 1.2 既存のアプローチ
この階層性を説明する試みは、伝統的に動的対称性の破れ（テクニカラー）[3]、放射補正、あるいは余剰次元における幾何学的解（ランドール・サンドラム模型）[2] に依存してきた。弦理論は、コンパクト化されたカラビ・ヤウ多様体の幾何学から湯川結合を導出する枠組みを提供するものの [4]、特定の真空状態を選択する一意のメカニズムは依然として見つかっていない。

### 1.3 関連研究と差別化
Koorambas (2022) による最近の研究では、粒子質量の文脈でカタラン数を探求しているが、我々のアプローチは、組み合わせ論的な数列ではなく、双曲体積の基本単位である**「幾何学的カタラン定数 ($G$)」**を採用している点で根本的に異なる。さらに、Hopf群の文脈で「双曲体積階層」が最近示唆されたが（2025）、KSAUは、$2^k$ 行列式ルールを通じてPDGフェルミオン・スペクトルへの絡み目補空間の具体的な割り当てを説明する、初の具体的な**選択メカニズム（第3の原理）**を提供する。連続的な歪曲計量（Warped Metric）に依存するランドール・サンドラム模型 [2] とは異なり、KSAUは**離散的な位相不変量**を採用しており、より厳格でパラメータのない予測を提供する。

### 1.4 位相的動機
本研究では、フェルミオンのアイデンティティが結び目および絡み目補空間の3次元トポロジーにコード化されているという、独自の幾何学的アプローチを提案する。Wittenによって確立された位相的場の理論（TQFT）と結び目不変量の深い関係 [5]、およびMostowによって発見された双曲体積の剛性（Rigidity）[6] に動機付けられ、我々は粒子の「フレーバー」が特定の位相型に対応するかどうかを調査する。

## 2. 手法とデータ (Methods and Data)

### 2.1 実験データ
すべての粒子質量は、Particle Data Group (PDG) 2024 Review [1] から引用している。カレントクォーク質量（$\overline{MS}$ スキーム）の中央値およびレプトンのポール質量を使用する。

### 2.2 位相データベース
結び目および絡み目の不変量は、**KnotInfo** および **LinkInfo** データベース [7, 8] から照会された。双曲体積は SnapPy カーネル [9, 10] を使用して計算されている。

### 2.3 選択手順（厳格なフィルタリング）
過学習（数千の結び目からのチェリーピッキング）のリスクを排除するため、質量フィッティングを行う前に、厳格な**ア・プリオリ（先験的）な**フィルタリングアルゴリズムを適用した：
1.  **成分数フィルタ:** クォークには絡み目（Links）、レプトンには結び目（Knots）。
2.  **行列式フィルタ（第3の原理）:** ダウン型クォークには「二進行列式ルール（Binary Determinant Rule, $Det=2^k$）」を、レプトンには「素数/奇数ルール」を適用した。この位相的制約により、候補プールは4000以上の絡み目から、**各世代あたり5つ未満の候補**にまで減少した。
3.  **体積マッチング:** この高度に制限されたサブセット内でのみ、$| \ln(m_{obs}) - \ln(m_{pred}) |$ を最小化する候補を選択した。

この多段階のプロセスにより、位相的割り当てが主に離散的な不変量（行列式、成分数）によって駆動されることが保証される。我々のモデルは、**連続的な調整パラメータをゼロ**にしていることを強調する。係数 $\gamma$ と $b'$ は、数値回帰ではなく、幾何学的な第一原理（カタラン定数と次元性）によって固定されている。レプトンチャネルについては、電子の質量によって固定される1つの規格化定数 ($C_l \approx -2.503$) が、そのセクターの唯一の較正点として機能する。

### 4.1 クォークチャネル（7次元チャネル）

$$ \ln(m_q) = \left( \frac{10}{7} G ight) V_q - (7 + G) $$
ここで $V_q$ は絡み目補空間の双曲体積である。このモデルは、クォーク質量を **8.7%** のMAEで予測する。

## 5. 考察 (Discussion)

### 5.1 次元の分岐とRS模型との相違
KSAUによって示唆される7D/9Dの分岐は、ランドール・サンドラム（RS）のような従来の余剰次元モデルとは異なる [2]。RSでは、質量階層は連続的に歪曲したバルク内のブレーンの位置によって生成される。一方KSAUでは、階層は**離散的な位相チャネル**によって決定される：クォークは有効的な7次元コンパクト化幾何チャネルと整合し、レプトンは9次元投影チャネルと整合する。同じ双曲「張力」$G$ によって正規化されたこの分岐は、歪曲因子の「微調整（ファイン・チューニング）」の必要性を排除する。重要なことに、これらの次元は物理的な時空の拡張として仮定されるのではなく、離散的な位相不変量から推論される有効的な幾何学的チャネルとして現れる。

### 5.2 行列式ルールの物理的メカニズム
「二進行列式ルール」（ダウン型クォークで $Det = 2^k$）は、真空の基礎となる量子状態への深い関連を示唆している。3D-TQFTにおいて、結び目行列式 $|V(-1)|$ は理論のヒルベルト空間における状態数に関連している。$2^k$ の進行は、ダウン型クォークが**マルチ量子ビット（Multi-qubit）位相状態**としてモデル化できる可能性を示しており、各世代がリンク構造に情報の基本ビットを追加していることを示唆する。パリティの分離（クォークは偶数、レプトンは奇数）は、おそらくゲージ群の中心に関連した、整数電荷セクターと分数電荷セクターの分離に対する位相的な説明を提供する。

### 5.3 ミューオンの異常と重いクォークの偏差
電子とタウの適合は精密（<1.5%）であるが、ミューオンは+17.8%の偏差を示す。この系統的なオフセットは、$N=6$ セクターにおいて、単純な $N^2$ スケーリングでは捉えきれない特定の位相的補正（例えば、$6_1$ 結び目のツイスト数に関連するもの）が必要である可能性を示唆している。クォークセクターでは、ボトムクォーク（$-17.3\%$）とトップクォーク（$+13.1\%$）が最大の偏差を示している。ボトムは系統的に過小予測され、トップは過大予測されており、線形の $\ln(m) \propto V$ スケーリングが体積範囲の両端において高次の補正を必要とする可能性を示唆している。これは v4.1 における最も重要なターゲットであり、$V > 12$ のリンクに対してブリッジ数（Bridge Number）やライセ（Writhe）による補正項が必要になる可能性がある。

### 5.4 レプトンチャネル：なぜ体積ではなく $N^2$ なのか？
クォークチャネルとレプトンチャネルの間の重要な非対称性には説明が必要である。クォークの場合、質量公式に入る位相不変量はリンク補空間の双曲体積 $V$ である。一方、レプトンの場合は、代わりに交差数の二乗 $N^2$ を使用する。これは恣意的な選択ではなく、幾何学的な必然性である：電子は三葉結び目 $3_1 = T(2,3)$ に割り当てられているが、これは**トーラス結び目**であり、したがって**双曲的ではない**（$V_{hyp} = 0$）。最も単純な非自明な結び目が双曲構造を持たないため、レプトンチャネルは組織化不変量として双曲体積を使用することができない。幾何学的タイプに関係なくすべての結び目に対して定義される交差数 $N$ は、自然な代替手段を提供する。それにもかかわらず $N^2$ スケーリングが一貫した質量スペクトル（MAE = 6.3%）を生成するという事実は、交差数が非双曲領域におけるより深い幾何学的量の「組み合わせ論的プロキシ（代理変数）」として機能している可能性を示唆している。この非対称性を解決すること——おそらく、双曲多様体と非双曲多様体の両方に有効な統一不変量を通じて——は、将来の研究の中心的な目標である。

### 5.5 ボロミアン環と色の閉じ込め
本研究における決定的な発見は、ダウンクォークが正確に **$8G$** の双曲体積を持つリンク $L6a4$ に対応することである。これは、ダウンクォークのトポロジーを**ボロミアン環（Borromean Rings, $6^3_2$）**として特定する。これは深遠である：ボロミアン環は、どの2つのループも互いにリンクしていないが、3つ合わせると分離不可能になるという性質を持つ。これは、バリオン（$qqq$）内のクォークの**色の閉じ込め（Color Confinement）**に対する直接的な幾何学的類似物を提供する。そこでは、シングレット状態のために3つの色が必要であり、個々のクォークは分離できない。$8G$ の出現は、最も軽い3成分リンクに対する幾何学的な「量子化条件」として機能する。

## 6. 結論 (Conclusion)

我々は、標準模型のフェルミオン質量が、結び目理論に根ざした3つの幾何学的原理を通じて理解できることを実証した：
1.  **閉じ込め**は位相的な絡み合い（Linking）に対応する。
2.  **電荷**は行列式のパリティを決定する。
3.  **質量**はカタラン定数を介して双曲体積でスケーリングする。

この枠組みは、最大で1つの較正定数のみを用いて **7.9%** の全体精度 ($p < 10^{-5}$) を達成し、弦理論のコンパクト化に対する潜在的な候補選択原理と、SMを超えた物理学（Beyond SM Physics）に対する検証可能な予測を提供する。3次元トポロジーと4次元素粒子物理学の間の深い結びつきは、「フレーバー」が真空の幾何学的構造の現れである可能性を示唆しており、自然界の基本的なアーキテクチャを理解するための新たな道を切り開くものである。

## 補足資料 (Supplementary Material)

### S1. 候補選択の詳細
各粒子のトップ5候補は、第3の原理（行列式ルール）に基づいて評価された。すべてのケースにおいて、v4.0の割り当ては、許容される行列式パリティクラス内での誤差の唯一のグローバル最小値を表している。

重要な点として、ボトムクォークとトップクォークに関しては、行列式ルールが**より優れた質量適合をもたらす候補を棄却している**。ボトムクォークの場合、$L10a146$ (Det = 60, MAE = 4.7%) は、選択された $L10a141$ (Det = 64 = $2^6$, MAE = 8.4%) よりも大幅に優れた体積一致を提供するが、$60 
eq 2^k$ であるため棄却される。トップクォークの場合、$L10a107$ (MAE = 6.9%) は、体積のみで見れば $L11a62$（選択された候補）よりも優れている。これは、行列式ルールが真の予測内容を持っていることを示している。つまり、事前に選択された割り当てを事後的に正当化するのではなく、離散的な位相的一貫性を優先するために適合度（Goodness-of-fit）を犠牲にしているのである。

### S2. 統計的堅牢性

**置換検定 ($p < 10^{-5}$):** 質量とトポロジーの相関の統計的有意性は、ノンパラメトリックな置換検定（Permutation Test）によって評価された。観測された相関が偶然であるという帰無仮説の下で、$N = 10^5$ 回のランダムな位相割り当てが、許容される成分数プール（アップ型クォーク用2成分、ダウン型クォーク用3成分）内でリンク/結び目候補をシャッフルすることによって生成された。各ランダム割り当てに対して、回帰 $\ln(m) = \gamma V + b'$ を再適合し、$R^2$ を計算した。$10^5$ 回の置換のうち、$R^2 \ge 0.9992$（観測値）を達成したものは一つもなく、その結果 $p < 10^{-5}$ が得られた。

**ブートストラップ解析 ($N=10,000$):** 残差ブートストラップ法により、傾き $\gamma$ に対する95%信頼区間が得られた：
*   $\gamma_{theory} = 1.3085$
*   $\gamma_{95\% CI} = [1.28, 1.34]$

これは、幾何学的理論値が実験データの統計的境界内に十分に収まっていることを確認するものである。

**一個抜き交差検証 (LOO-CV):** LOO-CV は **14.9%** の平均絶対誤差をもたらし、これはインサンプルMAEの8.7%の約1.7倍である。この上昇はサンプルサイズが小さい（$n=6$）ことを考慮すれば予想されることであり、相関が統計的に本物であっても（$p < 10^{-5}$）、特定のパラメータ推定値には意味のある不確実性が含まれることを示している。幾何学的モデル (v4.0) は、回帰ではなく第一原理から $\gamma$ と $b'$ を固定することで、この過学習のリスクを低減している。

**モデル選択に関する注記 (LEEの緩和):** どこでも効果（Look-Elsewhere Effect, LEE）に対処するため、各フレーバーの探索空間は、質量の比較を行う**前に**、離散的な位相不変量（成分数および行列式のパリティ）によって制限されたことを強調する。これにより、モデル選択における有効自由度が数千から各スロットあたり5未満の候補にまで減少し、事後的な過学習（Post-hoc Over-fitting）のリスクが大幅に緩和される。
