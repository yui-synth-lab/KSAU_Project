## 1. v18.0: 重力のインピーダンス解釈 (The Foundation)

- **論理**: 重力は多様体上の情報処理の「混雑（インピーダンス）」であると定義。この $\kappa$（真空のスペクトル重み）が、後の宇宙網における「位相的張力（抑制力）」の強さを決定する普遍的な定数となった。

## 2. v19.0 - v20.0: 宇宙網の分岐法則 (Filament Dynamics)
- **核心**: フィラメント分岐数 $B_{eff} = 4$ の導入。

## 3. v21.0 - v22.0: 抑制過剰の発見 (The 1.8σ Crisis)
- **核心**: 線形成長モデルにおける小スケール乖離の顕在化。
- **論理**: 「一様な位相的張力」だけでは、DES や HSC は説明できても、小スケールを観測する KiDS-Legacy において成長を抑制しすぎてしまうことが判明。この **1.8σ の不一致**が、理論に「非線形性」を導入すべき数学的サインとなった。

## 4. v23.0: 誠実な統合と解決 (Honest Resolution)

- **論理**:

    - **幾何学的必然性**: 赤方偏移進化指数 $\beta = 13/6$ と、絡み合いブースト $1 + \alpha(k/\kappa)^2$ を導入。
    - **バリオンのSSoT化**: バリオン影響を外部パラメータではなく、E8ルート格子へのエントロピー流出（スケール 16.0）として幾何学的に定義。
